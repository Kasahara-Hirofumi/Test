**「貯蓄で吸収できるリスクには保険を掛けない」原則** を元に考えると、2人ともに最低限の保険に入るという選択肢が最善と判断する。

---

## 保険の型選び

上記を加味すると、保険の型は:

- 掛け捨て
- 平準型(最低限という原則より、後述)
- 逓減型(合理的にリスクをつぶせる)

---

## 保険の種類(加入イメージ)

**加入する保険: 世帯合計 約8,000〜13,000円/月(例)**

| 項目 | 内容 |
|---|---|
| 型 | 掛け捨て・平準型・逓減型・定期(55～60歳満了) |

## 富国生命さんのプラン(対面ch型)に関して

### ミスマッチと感じる部分

- 最低限の保険の場合、相談役の必要性が薄くなってしまう。
- 夫婦ともに加入する場合、同じような更新相談・手続きのために実質2倍相談料を払うことと同義となる。
  (もし片方だけ入る場合は手間が省略できるという利点を失う)
- 相見積もりで見た場合の価格差が大きい(約2倍弱、説明時に要確認)
  (収入保障保険のみシミュ総支払差分: 配当控除後 差額1,666円 × 12ヶ月 × 21年 ≒ 約42万円)

## まとめ

富国生命さんのプランは、一家の大黒柱の収入源に対してきめ細かく保険を掛けることを前提に組まれているプランだと感じる。
(具体的に言うと特約をきめ細かくつけた場合の手続きや判断の手間/リスクを割り増し保険料で買っている形。)

前述の「ミスマッチと感じる部分」は富国生命さんのプランが悪いというわけではなく、そもそも私たち夫婦のような収入・資産形態に対しての決定的なミスマッチが大元にあるのではないかという推察に落ち着いた。

## 参考情報: 保険を分解する5つの軸

| 軸 | 選択肢 | 意味 | 推奨 |
|---|---|---|---|
| ① お金の性質 | 掛け捨て | 保険料は保障の購入費。戻ってこない分、安い | |
| | 貯蓄型 | 解約返戻金・満期金あり。その分高く、利回りは低い | |
| ② 保険料の推移 | 平準型(=全期型) | 加入時の料率で満期まで固定。総額の見通しが立つ | |
| | 更新型 | 10〜15年ごとに区切り、更新時の年齢で再計算=値上がり | |
| ③ 保障額の推移 | 逓減型 | 保障額が時間とともに減る(収入保障保険がこれ) | ← 収入保障 |
| | 定額型 | 保障額が満期までずっと同じ(就業不能保険は通常これ) | ← 就業不能 |
| ④ 保障の期間 | 定期(有期) | 60歳満了など期限あり。必要な期間だけ買うので安い ||
| | 終身 | 一生涯続く。高い。貯蓄型とセットのことが多い | |
| ⑤ 販売チャネル | 対面ch(営業職員) | 富国など大手生保。人件費が保険料に乗る。パッケージ販売が主流 | |
| | ネット・代理店ch | FWD・オリックス等。同じ保障が安いことが多い。単品で選べる | ← 比較先 |
